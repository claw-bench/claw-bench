# Project Brief: Team File Sharing Web App

Build a secure web application over three months, starting 2025-01-06 and ending by 2025-04-04.

Phases:

1. Foundation and setup: architecture, CI/CD, infrastructure, schema, and design system.
2. Core development: authentication, file upload/download, frontend workflows, and API integration.
3. Advanced features: real-time collaboration, search, versioning, admin analytics, and audit logging.
4. Testing and launch: QA, security review, performance tuning, documentation, and production release.

Team: two backend engineers, two frontend engineers, one designer, one QA engineer, and one project manager.
